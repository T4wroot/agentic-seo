# ایجنتیک سئو (Agentic SEO) 🤖

> **اولین استاندارد جامع سئو برای ایجنت‌های هوشمند خودمختار**

پروژه **Agentic SEO** یک مهارت (Skill) متن-باز و جهان‌شمول است که طراحی شده تا به هر ایجنت هوش مصنوعی (مانند Antigravity، AutoGPT، Hermes یا Cursor) یاد بدهد چگونه وب‌سایت‌ها را بر اساس قوانین واقعی و عمیق سئو طراحی و بهینه‌سازی کند.

## قابلیت‌های کلیدی

- **موتور استدلال جامع**: یک اسکریپت سبک پایتون که به عنوان منبع حقیقت (Source of Truth) برای استخراج قوانین سئو عمل می‌کند.
- **قوانین تخصصی اصناف**: قوانین عمیق برای سایت‌های فروشگاهی، SaaS و وبلاگ‌ها (مثل مدیریت سلسله مراتب $H_1$ تا $H_6$).
- **ضد الگوها (Anti-Patterns)**: لیستی از اشتباهات مرگبار سئو که ایجنت‌ها تحت هر شرایطی باید از آن‌ها دوری کنند (مثل حلقه‌های کنونیکال، مشکل CLS و آدم‌خواری کلمات کلیدی).
- **دانش اختصاصی فریم‌ورک‌ها**: بهترین شیوه‌ها برای Next.js، Nuxt و وردپرس.
- **یکپارچه‌سازی با ابزارهای مانیتورینگ**: اتصال مستقیم به Google Search Console و ابزارهای تحلیل لاگ برای بازخورد لحظه‌ای به ایجنت‌ها.

## ساختار پروژه

```text
agentic-seo/
├── src/agentic_seo/
│   ├── data/                  # دیتابیس‌های CSV شامل قوانین حرفه‌ای سئو
│   └── scripts/search.py      # موتور استدلال پایتون
├── cli/                       # ابزار خط فرمان Node.js
├── templates/                 # قالب‌های کانفیگ ایجنت‌ها
└── skill.json                 # مانیفست پلاگین
```

## نصب و استفاده

### 🤖 روش ایجنتیک (پیشنهاد ما)
شما حتی نیازی به نصب دستی این مهارت ندارید! کافیست پرامپت زیر را مستقیماً به ایجنت خود (مثل Antigravity یا Cursor) بدهید:
> *"لطفاً مخزن `https://github.com/tawroot/agentic-seo` را کلون کن، آن را به عنوان یک مهارت (Skill) روی خودت نصب کن و از این به بعد در تمام تسک‌های طراحی سایت، قوانین سئوی آن را رعایت کن."*

### 💻 روش دستی
ابتدا وارد پوشه خط فرمان شوید و پیش‌نیازها را نصب کنید:

```bash
cd cli
npm install
npm link # اختیاری: برای استفاده گلوبال از دستور agentic-seo
```

### دستورات CLI

**پیکربندی برای یک ایجنت خاص:**
```bash
node index.js init --ai antigravity
```

**جستجوی قوانین بر اساس صنف سایت:**
```bash
node index.js search --industry e-commerce
```

**مشاهده اشتباهات مرگبار (Anti-patterns):**
```bash
node index.js search --anti-patterns
```
