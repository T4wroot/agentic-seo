ویژگی‌های اصلی

🧠

موتور استخراج قوانین

اسکریپت پایتون برای خواندن داده‌های CSV و ارائه ساختاریافته‌ی اطلاعات سئو به ایجنت‌های متصل به صورت کاملاً خودکار.

🏢

قوانین اختصاصی

دستورالعمل‌های سئو برای سایت‌های فروشگاهی، SaaS و وبلاگ‌ها، شامل استانداردهای سلسله مراتب هدینگ‌ها و Core Web Vitals.

🚫

ضد الگوها (Anti-Patterns)

مرجعی از خطاهای رایج سئو مانند حلقه‌های کنونیکال تا ایجنت‌ها بتوانند از پیاده‌سازی روش‌های نادرست جلوگیری کنند.

📝

۱۶۰+ فایل مهارت Markdown

کتابخانه‌ای عظیم شامل بیش از ۱۶۰ مهارت متنی برای سئو و محتوا، که به صورت بومی و مستقیم توسط Cursor و Claude Code قابل خواندن است.

⚙️

۸۹ اسکریپت پیشرفته پایتون

شامل ۸۹ اسکریپت پایتون، ۱۰ ایجنت متخصص و نصب‌کننده‌های خودکار برای تمام محیط‌ها (Antigravity, Cursor, Windsurf و...).

نحوه کارکرد مهارت

۰۱

تزریق مهارت به ایجنت

ابتدا با استفاده از خط فرمان، فایل‌های پیکربندی و دانش سئو را به محیط ایجنت هوش مصنوعی خود (مانند Antigravity) معرفی می‌کنید.

۰۲

درخواست و استدلال

ایجنت پیش از نوشتن یا ویرایش کدهای HTML، از موتور استدلال پایتون درخواست می‌کند تا قوانین سئوی مربوط به صنف سایت را به او بدهد.

۰۳

تولید کدهای استاندارد

ایجنت قوانین ساختاریافته (مثل استفاده از دقیقاً یک تگ H1 و رعایت اسکیما مارک‌آپ) را روی کدها اعمال کرده و یک سایت ۱۰۰٪ بهینه تحویل می‌دهد.
